📚 Sobre o Projeto

Este é um trabalho acadêmico para o TCC sobre o comportamento das plataformas de apostas

🎯 Objetivo

Demonstrar como as plataformas de apostas utilizam a sensação inicial de ganho para incentivar os usuários a continuarem apostando, levando a perdas progressivas ao longo do tempo.

🔬 Metodologia

A simulação utiliza probabilidades variáveis para replicar o comportamento real das plataformas, dividindo a experiência em três fases distintas:

  • ✅ Confiança: Altas probabilidades de vitória para criar confiança
  • ⚖️ Equilíbrio: Resultados mistos para manter o engajamento
  • 📉 Desvantagem: Baixas probabilidades onde a casa prevalece
